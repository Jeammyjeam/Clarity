# **App Name**: Clarity AI

## Core Features:

- Mood Scanning: Analyze user's facial expressions using the camera to determine their current emotional state.
- Affirmation Generation: Provide a personalized affirmation to help improve the user's mood. Uses a tool to avoid platitudes and repetitions.
- Mood Visualization: Display the identified mood using clear and intuitive visualizations.
- Mood Journaling: Allow users to log their moods and track their emotional journey over time.
- Breathing Exercises: Offer a selection of guided breathing exercises tailored to different emotional states.
- Content Curation: Show curated articles and wisdom quotes based on current mood. Filter content to match detected mood.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to inspire feelings of calmness and serenity.
- Background color: Light grayish-blue (#D3E5EF), very desaturated to promote focus and avoid distraction.
- Accent color: Pale lavender (#D8B4F3), providing a gentle, reassuring highlight, especially for interactive elements.
- Body and headline font: 'PT Sans' for readability and a touch of warmth. 'PT Sans' is a humanist sans-serif, for a modern look.
- Use soft, rounded icons that represent various emotions and activities.
- Maintain a clean and minimalist layout with ample spacing for clarity and ease of use.
